The Chalkboard Library

Welcome

Chalkboard is a JavaScript library developed by Zushah that provides a plethora of both practical and abstract mathematical functionalities for its user.

This website has detailed information on Chalkboard, instructions on how to install it, the full documentation for all of its features, and several example programs that make some interesting use of it.

Features

  • Many number-theory-related operations
  • Real-valued functions and their operations
  • Complex numbers and quaternions
  • Plotting functions, vectors, and matrices
  • Lots of geometric formulae
  • All 24 (both circular and hyperbolic) trigonometric functions
  • Many statistical operations on arrays
  • Two-, three-, and four-dimensional vectors
  • Several-dimensional matrices
  • Calculus operations

And it's all packed into a single namespace written in a single 0.1 megabyte (0.05 megabyte minified) file.
