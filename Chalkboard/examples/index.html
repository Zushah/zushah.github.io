Chalkboard Examples

Here are a few interesting examples that demonstrate some of the features of Chalkboard (their source code can be found here):

  • Fluid flow
  • Hyperbolic functions
  • Isomorphism visualization
  • Modular multiplication mandala
  • Mandelbrot set
  • Matrix donut
  • Newton's method
  • Quaternion donut
  • Telemetry dashboard
  • Three-body problem

There is also one example that is more in-depth, ODE Solver Error vs. Step Size Study, which performs a benchmarking study of the error vs. step size of various fixed-step methods for numerically solving ordinary differential equations, so it shows Chalkboard's support for not only cool visualizations but also computational demos in numerical analysis.
