The Chalkboard Library

About

The Chalkboard library and this website were both created by Zushah on GitHub and Khan Academy.

The current latest release of Chalkboard is version 1.4.0 which was released on December 11, 2023. So far, Chalkboard has a total of 380 commands. It's available for regular JavaScript and also for Processing.js, too (which is compatible with the Khan Academy computer programming environment).

If you wish to contribute to the Chalkboard repository, you can follow these steps:

  1. Fork Chalkboard's GitHub repository. Make sure the fork is based on the main branch.
  2. Make your contributions to your fork. If you're adding new features or modifying the functionalities of old ones, please make sure you do the same for both editions (JavaScript and Processing.js) of the library.
  3. When you're done, commit the changes to your fork with a detailed description.
  4. Open a pull request. It will be reviewed soon and then merged to the main branch.

Thanks to Bhavjit Chauhan for his contribution to v1.3.0. Thanks to G. Yang for his contribution to the documentation.
