The Chalkboard Library

Installation

There are a couple different ways to install the different editions of Chalkboard into your project.

If your JavaScript program is being run within a webpage, the most straightforward way to import the Chalkboard library is to copy-and-paste this HTML line into your webpage:

<script type="text/javascript" src="https://cdn.jsdelivr.net/gh/Zushah/Chalkboard@1.1.0/src/Chalkboard.min.js"></script>

Otherwise, you can download the latest release of Chalkboard and put the relevant files in the directory of your project.

To download the Processing.js edition of Chalkboard, you will have to copy-and-paste this line (intsead of the one above) into your webpage:

<script type="text/javascript" src="https://cdn.jsdelivr.net/gh/Zushah/Chalkboard@1.1.0/src/ChalkboardProcessing.min.js"></script>

In a Khan Academy Processing.js program, unfortunately the only option you will have is to copy-and-paste the whole library into your program from the Khan Academy release.
