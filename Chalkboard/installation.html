The Chalkboard Library

Installation

There are a few different ways to install the different editions of Chalkboard into your project.

If your JavaScript project is being run on the client-side within a webpage, you can install Chalkboard via jsDelivr with this HTML tag:

<script src="https://cdn.jsdelivr.net/gh/Zushah/Chalkboard@1.7.0/src/Chalkboard.min.js"></script>

If your JavaScript project is being run on the server-side within the Node.js environment, you can install Chalkboard via npm with this console command:

npm install @zushah/chalkboard@1.7.0

Alternatively, you can simply download the latest release of Chalkboard and put the relevant files in your project's directory.

To install the Processing.js edition of Chalkboard, you will have to copy-and-paste this HTML tag (instead of the one above) into your webpage:

<script src="https://cdn.jsdelivr.net/gh/Zushah/Chalkboard@1.7.0/src/ChalkboardProcessing.min.js"></script>

In a Khan Academy Processing.js program, the only option you will have is to copy-and-paste the whole library into your program from the Khan Academy release.
