The Chalkboard Library

Documentation

This page has all the information you need to successfully operate the Chalkboard library. If you have any questions, feel free to contact Zushah.

Click on a category below to view the documentation for each of its commands.

The source code for the latest release can be found here.


Data Types

func

DESCRIPTION

The type used for mathematical functions in Chalkboard commands.

PROPERTIES

Definition {string} - The definition of the function
Type {string} - The type of the function, which is either "expl", "pola", "curv", "surf", or "mult"

EXAMPLE

// The variable f is of type func
let f = Chalkboard.real.function("x * x");

DESCRIPTION

The type used for complex numbers.

PROPERTIES

a {number} - The value of the real part of the complex number
b {number} - The value of the imaginary part of the complex number

EXAMPLE

// The variable z is of type comp
let z = Chalkboard.comp.new(1, 2);

real

Chalkboard.real.function(definition, type)

DESCRIPTION

Defines a mathematical function with either an explicit single-variable representation, a polar one, a parametric curve one, a parametric surface one, or an explicit multivariable one.

PARAMETERS

definition {string | string[]} - The mathematical expression that defines the function, which is written as a string for explicit, polar, and multivariable functions, as an array of two or three strings for parametric curves, and as an array of three strings for parametric surfaces. The variable used for explicit functions must be "x", the variable used for polar functions must be "O", the variable used for parametric curves must be "t", the variables used for parametric surfaces must be "s" and "t", and the variables used for multivariable functions must be "x" and "y".
type {string} (default = "expl") - The type of the function, which can only be either "expl" for an explicit function, "pola" for a polar function, "curv" for a parametric curve, "surf" for a parametric surface, and "mult" for a multivariable function

RETURNS

{func} - A new Chalkboard function defined by the definition parameter

EXAMPLE

// Defines the explicit single-variable function f(x) = x^2
let f = Chalkboard.real.function("x * x");

// Defines the polar function r(O) = cos(O)
let r = Chalkboard.real.function("Math.cos(O)", "pola");

// Defines the parametric curve r(t) = (cos(t), sin(t))
let r = Chalkboard.real.function(["Math.cos(t)", "Math.sin(t)"], "curv");

// Defines the parametric curve r(t) = (cos(t), sin(t), t)
let r = Chalkboard.real.function(["Math.cos(t)", "Math.sin(t)", "t"], "curv");

// Defines the parametric surface r(s, t) = (cos(s)cos(t), cos(s)sin(t), sin(s))
let r = Chalkboard.real.function(["Math.cos(s) * Math.cos(t)", "Math.cos(s) * Math.sin(t)", "Math.sin(s)"], "surf");

// Defines the explicit multivariable function f(x, y) = x^2 + y^2
let f = Chalkboard.real.function("x * x + y * y", "mult");

Calculates the curve length or surface area of a parametric curve or surface Chalkboard function within particular intervals.

PARAMETERS

func {func} - The parametric curve or surface Chalkboard function to integrate on
a {number} - The lower bound of the t-interval
b {number} - The upper bound of the t-interval
c {number} - The lower bound of the s-interval
d {number} - The upper bound of the s-interval

RETURNS

{number} - The arc length or surface area

EXAMPLE

// Returns 4π
let r = Chalkboard.real.function(["Math.cos(s) * Math.cos(t)", "Math.cos(s) * Math.sin(t)", "Math.sin(s)"], "surf");
let fds = Chalkboard.calc.fds(r, 0, Chalkboard.PI(2), Chalkboard.PI(-1/2), Chalkboard.PI(1/2));
